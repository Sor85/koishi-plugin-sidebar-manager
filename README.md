# koishi-plugin-sidebar-manager

Koishi 控制台侧栏管理插件。
用于收纳、排序和调整控制台侧栏项目的位置。

## 功能

- 查看当前控制台侧栏项目。
- 搜索侧栏项目名称、描述或 ID。
- 将侧栏项目收纳到侧栏管理分组。
- 调整侧栏项目顺序。
- 在顶部侧栏和底部侧栏之间移动项目。
- 重置侧栏布局。

## 安装

在 Koishi 插件市场搜索并安装：

```text
sidebar-manager
```

也可以通过包管理器安装：

```bash
npm install koishi-plugin-sidebar-manager
```

## 使用

1. 在 Koishi 控制台启用 `sidebar-manager` 插件。
2. 打开控制台侧栏中的“侧栏管理”页面。
3. 按需收纳、排序或移动侧栏项目。
